# BrandTide

BrandTide is a code-mixed sentiment analysis SaaS platform designed specifically for e-commerce reviews in Pakistan. It processes both Roman Urdu and English text to provide actionable sentiment intelligence for brands and products.

## Features

* Single and bulk CSV review sentiment classification.
* Code-mixed language support (English and Roman Urdu).
* Real-time dashboard for sentiment trends and distribution.
* Automated brand and product metric tracking.
* PDF report generation and automated email scheduling.

## Tech Stack

* **Frontend:** React, TypeScript, Tailwind CSS, Vite
* **Backend:** Node.js, Express, MongoDB
* **Machine Learning:** Python, FastAPI, XLM-RoBERTa
* **Deployment:** Vercel (Frontend), Render (Backend and ML Service)
